A key feature of this theme is its support for IndieWeb features, including microformats and web sign-in.

Web sign-in

Indigo handles web sign-in by setting the authorization_endpoint to IndieAuth.com:

microformats

Indigo marks up content with appropriate microformats, which provides semantic definitions of your content to other software. Posts are marked up with h-entry classes, like p-name, p-author, and e-content, while the author bio is marked up with h-card classes, including u-photo, u-url, p-locality/p-country-name, and p-note.

Open typefaces

Indigo uses a combination of three beautiful typefaces to render your words.

  • Fira Sans for heading text
  • Charter for body text
  • Fira Code for monospaced text

Licenses are included in the theme‚Äôs static/fonts folder.
